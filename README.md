# mux-remotion-demo

Make sure to set your Mux access token and secret key!

Copy the `.env.example` file to a new file called `.env` and add your keys there.

## Commands

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Server render demo**

```console
npm run server
```

See [docs for server-side rendering](https://www.remotion.dev/docs/ssr) here.

**Upgrade Remotion**

```console
npm run upgrade
```
